# Cantonal Law Expansion Plan

Date: 2026-04-13

## Overview

Expand openlegalcommentary from 9 federal laws to comprehensive coverage of
cantonal law across all 26 cantons. Three categories: constitutions, environment,
and education — plus key substantive laws for major cantons.

## Content architecture

Flat structure with canton prefix:

```
content/
  bv/             # Federal
  zgb/
  ...
  zh-kv/          # Cantonal: {canton}-{law}
  zh-vsg/
  zh-energ/
  be-kv/
  be-vsg/
  ...
```

URL pattern: `/{lang}/{canton}-{law}/art-{number}`
Example: `/de/zh-kv/art-1`, `/fr/ge-lip/art-15`

## Source data pipeline

1. **lexfind.ch** — aggregates all cantonal Gesetzessammlungen, structured data
2. **Individual cantonal law databases** — zhlex.zh.ch, belex.sites.be.ch, rsv.vd.ch, etc.
3. **opencaselaw.ch** — cantonal court decisions (Verwaltungsgericht, Obergericht)

First step: verify lexfind.ch API availability for systematic article text fetching.

## Implementation sequence

1. Verify source data (lexfind.ch API)
2. Build article fetcher for cantonal laws
3. Scaffold content directories
4. Pilot: ZH KV (133 articles, ~CHF 800)
5. Validate quality with triple evaluator
6. Scale: remaining 25 KVs
7. Wave 2-4 in parallel batches
